# Binary Programmer Calculator

This is a JavaFX-based binary calculator created as part of the
ICS 1201 Object Oriented Programming project.

## Features
- Binary addition
- Binary subtraction
- Binary multiplication
- Binary division
- Windows-style programmer calculator interface
- Input restricted to binary digits (0 and 1)

## Technologies Used
- Java
- JavaFX
- IntelliJ IDEA

## OOP Concepts Demonstrated
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
